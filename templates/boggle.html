About my Boggle Solver Project

This Python project implements a Boggle solver, showcasing strong algorithmic design and recursive techniques. The solver accommodates both user-input custom boards and randomly generated ones. It efficiently handles user inputs for board type and minimum word length. Leveraging natural language processing, the solver identifies valid words on the Boggle board. Key skills demonstrated include algorithmic design, recursion, user input handling, file handling for word dictionary, randomization, and functional decomposition. This Boggle solver project not only highlights fundamental programming skills but also showcases the application of advanced concepts in solving real-world word search challenges.
